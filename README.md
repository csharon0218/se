# SceneRoute

K-콘텐츠 기반 외국인 맞춤형 한국 체험 추천 시스템

## 프로젝트 개요
SceneRoute는 K-드라마와 K-영화에 관심 있는 외국인 관광객을 위해,
선호 작품과 사용자 성향을 바탕으로 한국의 촬영지, 음식, 문화 체험 요소를
결합한 맞춤형 여행 코스를 추천하는 시스템입니다.

기존 관광 서비스가 일반적인 관광 정보 제공에 집중하는 반면,
본 프로젝트는 사용자가 작품 속 주인공처럼 한국을 경험할 수 있도록
감성적이고 몰입감 있는 여행 경험을 설계하는 것을 목표로 합니다.
