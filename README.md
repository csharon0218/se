# SyncSpace

클라우드 파일 공유 시스템 (미니 드라이브)

## 프로젝트 개요
SyncSpace는 조직 내부에서 파일을 안전하고 효율적으로 저장, 관리, 공유할 수 있도록 지원하는 클라우드 기반 파일 공유 시스템입니다.

현재 문서, 설계 파일, 보고서, 이미지 등 다양한 업무 자료가 이메일, 메신저, 개인 클라우드 등에 분산되어 관리되면서 파일 버전 혼선, 파일 탐색 비효율, 외부 공유 시 보안 문제 등이 발생하고 있습니다.

본 프로젝트는 이러한 문제를 해결하기 위해 파일의 중앙 집중 관리, 폴더 기반 정리, 사용자 간 파일 공유, 검색, 버전 관리, 권한 기반 접근 제어 기능을 제공하는 웹 기반 시스템을 구축하는 것을 목표로 합니다.
