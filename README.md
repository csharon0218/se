# SyncSpace

클라우드 파일 공유 시스템 (미니 드라이브)

## 프로젝트 개요
SyncSpace는 조직 내부에서 파일을 안전하고 효율적으로 저장, 관리, 공유할 수 있도록 지원하는 웹 기반 클라우드 파일 공유 시스템입니다.

현재 조직 내에서는 문서, 설계 파일, 보고서, 이미지 등 다양한 업무 자료가 이메일, 메신저, 개인 클라우드 등에 분산되어 관리되고 있습니다. 이로 인해 파일 최신 버전 혼선, 파일 탐색 시간 증가, 외부 공유 시 보안 문제 등이 발생할 수 있습니다. 본 프로젝트는 이러한 문제를 해결하기 위해 파일의 중앙 집중 관리, 폴더 기반 정리, 사용자 간 공유, 검색, 버전 관리, 권한 기반 접근 제어 기능을 제공하는 시스템을 제안합니다.

본 저장소는 소프트웨어공학 과제 수행을 위한 GitHub 저장소이며, 프로젝트 문서와 추후 구현 코드를 함께 관리하는 것을 목표로 합니다.

- **프로젝트명**: 클라우드 파일 공유 시스템 구축 프로젝트
- **개발 시스템명**: SyncSpace
- **프로젝트 유형**: 웹 기반 파일 관리 및 공유 시스템

- **프로젝트 목표**: 파일을 중앙에서 안전하게 관리하고, 빠르게 탐색하며, 권한에 따라 효율적으로 공유할 수 있는 시스템 구축
